# Dossier Assets - Peace Magazine

Ce dossier contient tous les assets (images, icônes, etc.) utilisés par le site Peace Magazine.

## Structure

```
assets/
├── images/          # Images du site
│   ├── og-image.jpg     # Image pour Open Graph (1200x630px)
│   ├── twitter-image.jpg # Image pour Twitter Card (1200x630px)
│   ├── logo.png         # Logo Peace Magazine
│   └── gallery/         # Images de la galerie (optionnel)
└── icons/           # Icônes du site
    └── favicon.ico      # Favicon du site
```

## Recommandations

1. **Qualité** : Utilisez des images haute résolution
2. **Optimisation** : Compressez les images pour le web
3. **Formats** : 
   - JPG pour les photos
   - PNG pour les logos et images avec transparence
   - ICO pour les favicons
4. **Tailles** : Respectez les dimensions recommandées
5. **Cohérence** : Maintenez la cohérence visuelle avec la palette dorée

## Utilisation

Les images sont référencées dans le code HTML/CSS avec des chemins relatifs :
- `./assets/images/og-image.jpg`
- `./assets/images/logo.png`
- `./assets/icons/favicon.ico`
